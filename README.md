# BG-NPC-TAXI
BG NPC Taxi è uno script per FiveM pensato per rendere il tuo server più realistico, immersivo e comodo per i player.

<img width="1672" height="941" alt="bgnpctaxi" src="https://github.com/user-attachments/assets/efcbd0ae-dbcb-4de5-9660-f69a01e9da84" />

https://www.youtube.com/watch?v=r3sBetyzVA4

Con un semplice comando, il giocatore può chiamare un taxi NPC che arriverà direttamente vicino alla sua posizione. Una volta salito sui sedili posteriori, basterà impostare un waypoint sulla mappa e il tassista partirà automaticamente verso la destinazione scelta.

Il sistema include una UI stile tassametro, calcolo del prezzo in base ai chilometri percorsi, pagamento automatico dal conto bancario, blip del taxi sulla mappa e gestione completa della corsa. Il taxi può essere annullato, il prezzo è configurabile e tutti i parametri principali possono essere modificati facilmente dal config.

Caratteristiche principali:
• Taxi NPC richiamabile tramite comando
• Destinazione tramite waypoint del player
• Autista NPC automatico
• UI tassametro inclusa
• Tariffa base + prezzo per km
• Pagamento tramite banca
• Controllo saldo minimo per chiamare il taxi
• Blip del taxi sulla mappa
• Sistema configurabile
• Compatibile con ESX e ox_lib

Perfetto per server roleplay che vogliono aggiungere un servizio taxi automatico, pulito e funzionale, senza dover dipendere sempre da player online nel lavoro taxi.

BG NPC Taxi porta nel tuo server un’esperienza semplice, utile e immersiva per ogni giocatore.
